[-] General.
	- Reference.
		android_usage_guide.txt

[-] Run SimpleJni app.
	- Change directory.
		${SWDT-Mobile_HOME}/android/basic/SimpleJni/app/src/main/java
	- Generate JNI interfaces.
		javah -jni -o ../cpp/ArithmeticJni.h com.sangwook.simplejni.ArithmeticJni
		javah -jni -o ../cpp/StringJni.h com.sangwook.simplejni.StringJni
	- Change directory.
		${SWDT-Mobile_HOME}/android/basic/SimpleLib/externallib/src/main/java
	- Generate JNI interfaces.
		javah -jni -o ../cpp/TrigonometricJni.h com.sangwook.simplejni.TrigonometricJni
	- Create & implement JNI source files.
		${SWDT-Mobile_HOME}/android/basic/SimpleJni/app/src/main/cpp/ArithmeticJni.cpp
		${SWDT-Mobile_HOME}/android/basic/SimpleJni/app/src/main/cpp/StringJni.cpp
		${SWDT-Mobile_HOME}/android/basic/SimpleLib/externallib/src/main/cpp/TrigonometricJni.cpp

	- Open SimpleJni & SimpleLib library module.
		${SWDT-Mobile_HOME}/android/basic/SimpleJni
		${SWDT-Mobile_HOME}/android/basic/SimpleLib
	- Make SimpleLib library module.
		This module does not contain app module, so it can not be run.
		Output files:
			${SWDT-Mobile_HOME}/android/basic/SimpleLib/externallib/build/outputs/aar/externallib-release.aar
				==> Build a release Android library module (AAR).
			${SWDT-Mobile_HOME}/android/basic/SimpleLib/externallib/build/outputs/aar/externallib-debug.aar
		==> AAR files contain class & so files.
			So class & so files are not needed to link with app projects.
			AAR files are only required.
	- Import a JAR/AAR package into SimpleJni project.
		'File -> New -> New Module... -> Import .JAR/.AAR Package'
			${SWDT-Mobile_HOME}/android/basic/SimpleLib/externallib/build/outputs/aar/externallib-release.aar
	- (Optional) edit setting files.
		In ${SWDT-Mobile_HOME}/android/basic/SimpleJni/settings.gradle:
			include ':app', ':externallib-release'
		In ${SWDT-Mobile_HOME}/android/basic/SimpleJni/app/build.gradle:
			dependencies {
				...
			    compile project(':externallib-release')
				...
			}
	- Make SimpleJni library module.
	- Run SimpleJni library module.
		${SWDT-Mobile_HOME}/android/basic/SimpleJni/app/build/outputs/apk/app-release.apk
		${SWDT-Mobile_HOME}/android/basic/SimpleJni/app/build/outputs/apk/app-debug.apk
