[-] General.
	- Site.
		https://developer.android.com/studio/intro/index.html

	- Native Development Kit (NDK).
		JNI_in_Android.txt

[-] Usage.
	- Build a release Android library module (AAR).
		https://stackoverflow.com/questions/27646262/how-to-create-a-release-android-library-package-aar-in-android-studio-not-deb

	- Link AAR or JAR as external module.
		https://developer.android.com/studio/projects/android-library.html

	- Sign apps.
		https://developer.android.com/studio/publish/app-signing.html

[-] Run SimpleJni app.
	- Change directory.
		${SWDT-Mobile_HOME}/android/basic/SimpleJni/app/src/main/java
	- Generate JNI interfaces.
		javah -jni -o ../cpp/ArithmeticJni.h com.sangwook.simplejni.ArithmeticJni
		javah -jni -o ../cpp/StringJni.h com.sangwook.simplejni.StringJni
		javah -jni -o ../cpp/TrigonometricJni.h com.sangwook.simplejni.TrigonometricJni
	- Move a JNI interface.
		Copy ${SWDT-Mobile_HOME}/android/basic/SimpleJni/app/src/main/cpp/TrigonometricJni.h to ${SWDT-Mobile_HOME}/android/basic/SimpleLib/externallib/src/main/cpp/TrigonometricJni.cpp.
	- Create & implement JNI source files.
		${SWDT-Mobile_HOME}/android/basic/SimpleJni/app/src/main/cpp/ArithmeticJni.cpp
		${SWDT-Mobile_HOME}/android/basic/SimpleJni/app/src/main/cpp/StringJni.cpp
		${SWDT-Mobile_HOME}/android/basic/SimpleLib/externallib/src/main/cpp/TrigonometricJni.cpp

	- Open SimpleJni & SimpleLib library module.
		${SWDT-Mobile_HOME}/android/basic/SimpleJni
		${SWDT-Mobile_HOME}/android/basic/SimpleLib
	- Make SimpleLib library module.
		This module does not contain app module, so it can not be run.
		Output files:
			${SWDT-Mobile_HOME}/android/basic/SimpleLib/externallib/build/intermediates/cmake
			${SWDT-Mobile_HOME}/android/basic/SimpleLib/externallib/build/outputs/aar
				==> Build a release Android library module (AAR).
	- Copy native binaries.
		Copy files from ${SWDT-Mobile_HOME}/android/basic/SimpleLib/externallib/build/intermediates/cmake to ${SWDT-Mobile_HOME}/android/basic/SimpleJni/app/jnilibs/bin. (???)
		Copy files from ${SWDT-Mobile_HOME}/android/basic/SimpleLib/externallib/build/intermediates/cmake to ${SWDT-Mobile_HOME}/android/basic/SimpleJni/app/build/intermediates/cmake. (???)
	- Copy AAR file.
		Copy files from ${SWDT-Mobile_HOME}/android/basic/SimpleLib/externallib/build/outputs/aar/externallib-release.aar	 to ${SWDT-Mobile_HOME}/android/basic/SimpleJni/externallib-release/externallib-release.aar.
	- (Optional) import AAR in SimpleLib library module.
		'File -> New -> New Module... -> Import .JAR/.AAR Package'
		${SWDT-Mobile_HOME}/android/basic/SimpleLib/externallib/build/outputs/aar/externallib-release.aar
	- (Optional) edit setting files.
		In ${SWDT-Mobile_HOME}/android/basic/SimpleJni/settings.gradle:
			include ':app', ':externallib-release'
		In ${SWDT-Mobile_HOME}/android/basic/SimpleJni/app/build.gradle:
			dependencies {
				...
			    compile fileTree(include: ['*.jar','*.so'], dir: 'libs')
			    compile project(':externallib-release')
				...
			}
	- Make SimpleJni library module.
	- Run SimpleJni library module.
