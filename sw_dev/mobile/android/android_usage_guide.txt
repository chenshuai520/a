[-] Usage.
	- Copy file.
		Desktop -> emulator.
			adb push <local> <remote>
				adb push foo.txt /sdcard/foo.txt
				adb push foo.txt /storage/emulated/0/foo.txt
		Emulator -> desktop.
			adb pull <remote> <local>
