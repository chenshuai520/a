[-] Building in Win32/Cgywin
	[ref] https://github.com/xianyi/OpenBLAS/wiki/How-to-use-OpenBLAS-in-Microsoft-Visual-Studio

	-. use quick build.
		${OPENBLAS_HOME}/quickbuild.win32
		${OPENBLAS_HOME}/quickbuild.win64

	-. build library.
		1) run cygwin.
		2) run quick build script.
			cd ${OPENBLAS_HOME}
			./quickbuild.win32
			./quickbuild.win64
		3) run Visual Studio Command Prompt.
		4) generate def file.
			a) dumpbin.exe 이용.
				dumpbin.exe /exports libopenblas.dll > libopenblas.def
				==> 생성된 def file을 그대로 사용할 수 없음.
					def file 형식에 맞게 수정해야 함.
			b) OpenBLAS library에서 생성한 def file 이용.
				${OPENBLAS_HOME}/exports/libopenblas.def
				==> Windows 환경에서 사용할 수 있는지 검증 필요.
		5) generate import library.
			cd ${OPENBLAS_HOME}/exports
			For 32-bit, lib.exe /machine:i386 /def:libopenblas.def
			For 64-bit, lib.exe /machine:X64 /def:libopenblas.def
			==> ${OPENBLAS_HOME}/exports directory에 OpenBLAS에서 생성한 def file & import file이 존재.
				이들을 Windows 환경에서 사용할 수 있는지 검증 필요.

	-. install library
		make PREFIX=./install install
