[-] Libraries
	- BLAS.
	- GotoBLAS.
		(Windows) MinGW를 이용해서 build 후 사용.
			REF [site] >> http://icl.cs.utk.edu/lapack-for-windows/lapack/
		Library files:
			libgoto2_penrynp-r1.13.a/lib
		Features:
			'cblas_'로 시작하는 함수와 시작하지 않는 함수 모두 제공.
				e.g.) _sgemm_, _cblas_sgemm
			'lapack_'로 시작하는 함수와 시작하지 않는 함수 제공.
				e.g.) _cheev
	- OpenBLAS.
		(Windows) Cygwin을 이용해서 build 후 사용.
		Library files:
			libopenblas.so/dll(libopenblas.a)
		Features:
			'cblas_'로 시작하는 함수와 시작하지 않는 함수 모두 제공.
				e.g.) _sgemm, _sgemm_, _cblas_sgemm
			'LAPACKE_'로 시작하는 함수 제공.
				e.g.) _LAPACKE_cheev
	- LAPACK.
		lapack-3.6.0 version의 shared library 사용.
		(Windows) MinGW를 이용해서 build 후 사용.
			REF [site] >> http://icl.cs.utk.edu/lapack-for-windows/lapack/
		Library files:
			libblas.so/dll(libblas.lib)
			liblapack.so/dll(liblapack.lib)
			libtmglib.so/dll(libtmglib.lib)
		Features:
			함수명 뒤에 _가 붙어있는 형태로 export.
			'cblas_'로 시작하지 않는 함수 제공.
				e.g.) _sgemm_
			'clapack_'로 시작하지 않는 함수 제공.
				e.g.) _cheev_
	- CLAPACK.
		clapack-3.2.1-CMAKE version의 static library 사용.
		(Windows) Visual Studio를 이용해서 build 후 사용.
		Library files:
			libf2c.a/lib
			blas.a/lib
			lapack.a/lib
			tmglib.a/lib
		Features:
			함수명 뒤에 _가 붙어있는 형태로 export.
			'cblas_'로 시작하지 않는 함수 제공.
				e.g.) _sgemm_
			'clapack_'로 시작하지 않는 함수 제공.
				e.g.) _cheev_
	- ATLAS.
		atlas3.9.72 version의 static library 사용.
		Library files:
			f77blas.a
			cblas.a
			lapack.a
			atlas.a
		Features:
			'cblas_'로 시작하는 함수 제공.
			'clapack_'로 시작하는 함수 제공.
	- Eigen.
		BLAS & LAPACK의 function 대부분을 가지고 있지 않음.
		Library files:
			eigen_blas.so/dll
			eigen_lapack.so/dll
		Features:
			'cblas_'로 시작하지 않는 함수 제공. (?)
			'clapack_'로 시작하지 않는 함수 제공. (?)
	- GSL.
		gsl-1.15 version의 static library 사용.
		Library files:
			cblas.a/lib
			cblas.so/dll
		Features:
			'cblas_'로 시작하는 함수 제공.
