[-] Libraries
	-. BLAS & LAPACK
		CLAPACK 사용하는 것이 편리.
	-. CBLAS
		clapack-3.2.1-CMAKE version의 static library 사용.
		1. blas.a/lib
			'cblas_'로 시작하지 않는 함수 제공.
			함수명 뒤에 _가 붙어있는 형태로 export.
				e.g.) _dgemm_
	-. CLAPACK
		clapack-3.2.1-CMAKE version의 static library 사용.
		1. lapack.a/lib
			'clapack_'로 시작하지 않는 함수 제공.
			함수명 뒤에 _가 붙어있는 형태로 export.
				e.g.) _dgbsv_
	-. GotoBLAS
	-. OpenBLAS
		1. 아래의 library 제공
			libopenblas.a
			libopenblas.so/dll
			'cblas_'로 시작하는 함수 제공.
			'LAPACKE_'로 시작하는 함수 제공.
	-. ATLAS
		atlas3.9.72 version의 static library 사용.
		1. 아래의 library 제공
			f77blas.a
			cblas.a
			lapack.a
			atlas.a
			'cblas_'로 시작하는 함수 제공.
			'clapack_'로 시작하는 함수 제공.
	-. GSL
		gsl-1.15 version의 static library 사용.
		1. 아래의 library 제공
			cblas.a/lib
			cblas.so/dll
			'cblas_'로 시작하는 함수 제공.
