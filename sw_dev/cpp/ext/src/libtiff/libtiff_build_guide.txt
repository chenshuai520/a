<<  Building  >>

[-] TIFF site로부터 file download
	-. http://www.remotesensing.org/libtiff/
[-] 압축 해제
[-] MSVC 환경변수 설정
	-. Visual Studio 6.0
		>< %VC6_ROOT%/Vc98/bin/vcvars32.bat 실행
		>< command
			vcvars32.bat > nul
	-. Visual Studio .NET 2003
		>< %VC71_ROOT%/Vc7/bin/vcvars32.bat 실행
		>< command
			vcvars32.bat > nul
[-] build library
	nmake -f Makefil.vc all
[-] install library & header
	-. resulting library files are installed into a subdirectory
		%LIBTIFF_ROOT%/libtiff/libtiff.lib
		%LIBTIFF_ROOT%/libtiff/libtiff.dll
		%LIBTIFF_ROOT%/libtiff/libtiff_i.lib
	-. resulting include files are installed into a subdirectory
		%LIBTIFF_ROOT%/libtiff/tiff.h
		%LIBTIFF_ROOT%/libtiff/tiffio.h
		%LIBTIFF_ROOT%/libtiff/tiffvers.h
