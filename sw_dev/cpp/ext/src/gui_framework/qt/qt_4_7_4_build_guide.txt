[-] Building (Win32)
	[ref] http://qt-project.org/doc/qt-4.7/install-win.htm
	[ref] ${QT_ROOT}/INSTALL

	0. execute "Visual Studio Command Prompt"
	1. copy Qt sources to Qt directory (?)
		Qt source를 directory 'd:\Qt\4.7.4'에 복사했다면
		'nmake install'을 이용해 Qt를 설치할 때 아래와 같은 directory에 설치됨.
			${INSTALL_ROOT}\Qt\4.7.4
		따라서, source directory를 'd:\Qt-4.7.4'과 같이 하는 것이 바람직함.
			${INSTALL_ROOT}\Qt-4.7.4
		==> ${INSTALL_PREFIX}를 사용한다면, source directory를 고려하지 않아도 됨.
	2. set PATH
		${QT_ROOT}/bin
		    to locate qmake, moc and other Qt tools
	3. (optional) set the environment variable, QMAKESPEC
		set QMAKESPEC=win32-msvc2010
	4. change directory to ${QT_ROOT}
	    cd /D ${QT_ROOT}
	5. configure
	    configure -platform win32-msvc2010 -opensource -prefix ${INSTALL_PREFIX}
	    configure -debug-and-release -platform win32-msvc2010 -opensource -arch windows -shared -ltcg -no-fast -exceptions -accessibility -stl -qt-sql-sqlite -no-openvg -graphicssystem raster -qt-zlib -qt-gif -qt-libpng -qt-libmng -qt-libtiff -qt-libjpeg -no-dsp -no-vcproj -incredibuild-xge -plugin-manifests -qmake -process -rtti -mmx -3dnow -sse -sse2 -no-openssl -no-dbus -phonon -phonon-backend -multimedia -audio-backend -webkit -script -scripttools -declarative -qt-style-windows -qt-style-windowsxp -qt-style-windowsvista -no-style-plastique -no-style-cleanlooks -no-style-motif -no-style-cde -no-style-windowsce -no-style-windowsmobile -no-style-s60 -native-gestures -prefix ${INSTALL_PREFIX}
	    	[ref] http://qt-project.org/wiki/Building_Qt_Desktop_for_Windows_with_MSVC
	    	INSTALL_PREFIX는 '\usr\local\Qt-4.7.4'과 같이 지정.
	    ==> reconfigure
	    	nmake confclean
	    	configure ...
	6. build
		nmake
	7. (optional) set the environment variable, INSTALL_ROOT
		만약 install root directory로 'd:\usr\local'을 설정하고 싶다면
		drive에 대한 정보를 빼고 아래와 같이 설정.
			set INSTALL_ROOT=\usr\local
		설치되는 drive는 nmake가 실행되는 drive인 듯함.
		==> ${INSTALL_ROOT} 사용 보다 ${INSTALL_PREFIX}의 사용이 더 바람직함.
	7. install
		nmake install
			${INSTALL_PREFIX}/include, lib, & bin, etc에 설치.
			${INSTALL_ROOT}를 설정하는 경우 ${INSTALL_ROOT}/${INSTALL_PREFIX}/include, lib, & bin, etc에 설치.

[-] Trouble Shooting
	-. Q_INTERFACES "Error: Undefined interface"
		[ref] https://bugreports.qt-project.org/browse/QTBUG-16293
	-. manifest file 생성 error
		MSVC를 이용해 building하는 경우 mt.exe를 이용해 manifest file을 생성하는 과정에서
		manifest file을 생성하지 못하는 오류 발생.
		=> 다시 building하면 됨.
			원인을 알 수 없지만 Visual Studio IDE에서도 동일한 현상이 발생함.
