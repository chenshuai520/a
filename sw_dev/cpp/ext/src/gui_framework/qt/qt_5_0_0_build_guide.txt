[-] Building
	[ref] http://qt-project.org/doc/qt-5.0/install-win.html
	[ref] ${QT_HOME}/README & ${QT_HOME}/qtbase/INSTALL
	
	0. execute "Visual Studio Command Prompt"
	1. set PATH variable
		${PERL_HOME}/bin
		${PYTHON_HOME}
	2. configure
	    configure -prefix ${QT_INSTALL} -opensource
	3. nmake
	4. nmake install
		${QT_INSTALL}/bin/qtbase/qtbase에 설치됨
		환경변수, ${INSTALL_ROOT}를 설정하는 경우 ${INSTALL_ROOT}/${QT_INSTALL}/bin/qtbase/qtbase에 설치됨.

[-] Trouble Shooting
	-. manifest file을 생성 error
		MSVC를 이용해 building하는 경우 mt.exe를 이용해 manifest file을 생성하는 과정에서
		manifest file을 생성하지 못하는 오류 발생.
		=> 다시 building하면 됨.
			원인을 알 수 없지만 Visual Studio IDE에서도 동일한 현상이 발생함.
