<<  Building  >>

[-] downloads files from wxWidgets site
	-. http://www.wxwidgets.org
[-] builds a project
	<-> ver. 2.6 & msvc
		-. in Visual Studio IDE
			-> target project file
				"%WXWIDGETS_ROOT%/build/msw/wx.dsw"
				"%WXWIDGETS_ROOT%/build/msw/wx_dll.dsw"
			-> there are project dependencies
				0: wxexpat.lib wxjpeg.lib wxpng.lib wxregex.lib wxtiff.lib wxzlib.lib
				1: wxbase.lib
				2: wxmsw_core.lib
				3: wxbase_net.lib wxbase_odbc.lib wxbase_xml.lib
				4: wxmsw_adv.lib wxmsw_gl.lib wxmsw_html.lib wxmsw_media.lib wxmsw_qa.lib wxmsw_xrc.lib
				5: wxmsw_dbgrid.lib
		-. in CLI
			-> target project file
				"%WXWIDGETS_ROOT%/build/msw/"
			-> command
			 	nmake -f makefile.vc
			 	nmake -f makefile.vc clean
