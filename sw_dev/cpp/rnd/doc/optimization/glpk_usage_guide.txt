[-] Building (Windows).
	- WinGLPK 4.57.
		Build:
			Open the general Command Prompt, but not Visual Studio Command Prompt.
			cd ${WINGLPK_HOME}
			Run Build_WinGLPK.bat
		Output directory:
			Win32:
				${WINGLPK_HOME}/build32/w32
			Win64:
				${WINGLPK_HOME}/build64/w64
