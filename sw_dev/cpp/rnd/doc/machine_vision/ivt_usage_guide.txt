[-] Building (Linux).
	- Modify makefile.
		Set options below in makefile in order to use OpenGL, OpenCV, Video Capture (V4L, 1394, QuickTime).
			${IVT_HOME}/src/Makefile.base.
		<before>
			USE_OPENGL = 0
			USE_OPENCV = 0
			USE_OPENCV_HIGHGUI = 0
			USE_QUICKTIME = 0
			USE_LIBDC1394_V2 = 0
			USE_V4L = 0
		<after>
			USE_OPENGL = 1
			USE_OPENCV = 1
			USE_OPENCV_HIGHGUI = 1
			USE_QUICKTIME = 1
			USE_LIBDC1394_V2 = 1
			USE_V4L = 1
		==> Generated library.
			libivtvideocapture.a
		==> Can use OpenCV only.
			USE_OPENCV = 1
			USE_OPENCV_HIGHGUI = 1
	- Run make.
