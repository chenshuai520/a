[-] Installation (Windows).
	- Prepare the installation of GnuWin32.
		Download GetGnuWin32.
			http://getgnuwin32.sourceforge.net
		Run GetGnuWin32.
			GetGnuWin32-0.6.3.exe
			Destination folder: D:/depot/downloaded_material/util_software

	- Install GnuWin32.
		Run command prompt.
		Change directory.
			cd ${GetGnuWin32_HOME}
		Run download.
			download.bat
		Install GnuWin32.
			install.bat ${GNUWIN32_INSTALL_DIR}
