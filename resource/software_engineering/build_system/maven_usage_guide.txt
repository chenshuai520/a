[-] Usage.
	- Reference.
		http://javacan.tistory.com/129
		http://kjunine.tistory.com/35
		http://www.gliderwiki.org/wiki/105

	- Repositories.
		Remote repository: http://repo1.maven.org/maven2/
		Local repository: ${USER_HOME}/.m2/repository
			[groupId]/[artifactId]/[version]

	- Local directory structure.
		https://maven.apache.org/guides/introduction/introduction-to-the-standard-directory-layout.html

	- Create a project.
		mvn archetype:generate
			Generate pom.xml and directories.
		mvn archetype:create

		mvn dependency:copy-dependencies
			Copy all libraries and dependencies to the 'target/dependency' folder.

	- Build.
		mvn compile
		mvn -e compile
		mvn -X compile

		mvn test
		mvn clean

	- Package.
		mvn package
			Compile, run tests, and create jar.
		mvn package -Dmaven.test.skip=true
			Run no test.
		mvn deploy
